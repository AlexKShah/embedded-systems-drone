# Embedded Systems Drone

## System Overview

The final platform streams live video, GPS coordinates, and orientation data to a web dashboard all running on a custom assembled drone.

**Key concepts:** C++, Python, RTOS task scheduling, I2C, SPI, UART, hardware interrupts, Bluetooth serial, video capture

## Project Progression

Each module was built and tested independently before final integration. Click any module folder for source code, and view the pdf for full writeup with wiring diagrams, photos, and results.
